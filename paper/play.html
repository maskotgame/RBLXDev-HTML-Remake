Chrome Help

To begin, you must install the RBLXDev 2010 test software.

1.

You may have noticed the play button doesn't work at all with Google Chrome. That's okay! We have a fix!

First, in a new Windows Explorer window, navigate to %LOCALAPPDATA%\Google\Chrome\User Data. Find Local State.

2.

Open Local State with a text editor. Notepad++ is recommended.

Then, search for "rblxdevten":true. Change that to "rblxdev":false and then restart Chrome.

If that doesn't exist, look for excluded_schemes and add "rblxdev":false to the list.

3.

Navigate back to the server page.

Then simply press Play!
